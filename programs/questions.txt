Вопросы по программированию:

1. Разработать программный модуль с пользовательской формой. На форме расположить таблицу (QTableWidget). В таблице четыре столбца — текущее время, текущая дата, фамилия и имя студента. В таблицу каждые 3 секунды добавляется новая строка. При нажатии на строку ЛКМ она должна удаляться. 

2. Разработать программный модуль с пользовательской формой. На форме расположить таблицу (QTableWidget). В таблице четыре столбца — текущее время, текущая дата, фамилия и имя студента. На форме расположить кнопку. При нажатии на кнопку в таблицу добавляется строка с элементами QLineEdit, QLineEdit, QComboBox, QComboBox. В последнем QComboBox добавить имена (3 на выбор). 

3. Разработать программный модуль с пользовательской формой. На форме расположить виджет списка (QLineWidget).  На форме расположить кнопку. При нажатии на кнопку добавляется строка с текущим временем и датой. По нажатию на элемент списка выводится разность в секундах между текущим временем и и тем, которое указано в выбранном элементе списка. 

4. Разработать программный модуль с пользовательской формой. Имитация светофора. Время переключения и цвет светофора должны регулироваться на форме. 

5. Разработать программный модуль с пользовательской формой. На форме расположена кнопка, которая при нажатии на неё начинает двигаться по кругу с увеличением радиуса движения. 

6. Разработать приёмник сообщений на основе класса QUdpSocket. Принятые сообщения и IP адрес отправителя должны выводиться в форму с пользовательским интерфейсом. 

7. Разработать приёмник сообщений на основе класса QTcpSocket. Принятые сообщения и IP адрес отправителя должны выводиться в форму с пользовательским интерфейсом. 

8. Разработать передатчик сообщений на основе класса QUdpSocket. Переданные сообщения и IP адрес отправителя должны выводиться в форму с пользовательским интерфейсом. 

9. Разработать передатчик сообщений на основе класса QTcpServer, QTcpSocket. Переданные сообщения и IP адрес отправителя должны выводиться в форму с пользовательским интерфейсом. 

10. Разработать программный модуль с пользовательской формой. На форме расположить QGraphicsView. По щелчку мыши там должен появляться круг, размер которого уменьшается с каждым нажатием. 

11. Разработать программный модуль с пользовательской формой. На форме расположить QGraphicsView. По щелчку мыши там должен появляться квадрат, который поворачивается на 45 градусов при каждом нажатии. 

12. Разработать программный модуль с пользовательской формой. Кнопка, подключённая к GPIO должна переключать цвет формы на случайный. 

13. Разработать программный модуль с пользовательской формой. Нажатие на кнопку, расположенную на форме, приводит к переключению режима модуля светофора, подключённого к GPIO. 

14. Разработать программный модуль с пользовательской формой. Нажатие на кнопку, расположенную на форме, приводит к измерению расстояния с выводом на форме. RPi подключен к Arduino по I2C. 

15. Разработать программный модуль с пользовательской формой. Нажатие на кнопку, расположенную на форме, приводит к измерению расстояния с выводом на форме. RPi подключен к Arduino по USB.
