Установка необходимых для работы библиотек 
sudo apt install qt5-qmake qtbase5-dev qtchooser
sudo apt-get install libserialport-dev
Подключите Raspberry Pi к Arduino с помощью USB-кабеля.
Проверьте, что Arduino подключена к Raspberry Pi через последовательный порт. Обычно это будет /dev/ttyUSB0 или /dev/ttyACM0.
Создать кнопку measureButton и лейбл distanceLabel. 
Триггер подключить к 12, а эхл к 13 порту Ардуино. 
Библиотека на Arduino <NewPing.h>. 
Убедитесь, что последовательный порт правильно настроен в обоих приложениях.
Если ваш порт имеет другое имя (например, /dev/ttyACM0), измените его в коде.
На стороне Arduino используется датчик ультразвукового расстояния (HC-SR04), но можно использовать любой другой датчик по вашему выбору.
